HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The term "elements" represents the content between the angled brackets.
  • The angled brackets are called "tags".
  • Different tags are needed to represent the different types of components on a webpage.

CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicated how much space we want around the content inside an element.
  • A period (.) preceding a class name in CSS indicates that it will affect all classes with that name.
  • You can group multiple tags together in CSS if they are intended to have to same changes made.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: will copy the remote code locally and will notify if changes have been added or up to date.
  • git push origin branch-name: will send local code to the remote repo as a PR.
